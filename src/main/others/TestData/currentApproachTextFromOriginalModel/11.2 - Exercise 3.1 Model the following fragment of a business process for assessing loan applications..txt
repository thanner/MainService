The process starts when the Resource 1 prepares an acceptance pack. Next, it sends the acceptance pack and then receives the signed documents. Subsequently, it verifies the repayment agreement. After that, one of the 2 alternative procedures is executed. If "applicant agree", the Resource 1 approves the application. If "applicant disagree", the Resource 1 cancels the application. In any case, the Resource 1 notifies the application status. Finally, the process ends.
