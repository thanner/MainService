The process starts when the Resource 1 shortlists an application. Next, it conducts the activity. The subprocess contains the following steps. One of the 3 alternative procedures is executed. In the first procedure, the Resource 1 tests for drug and alcohols. In the second procedure, the Resource 1 checks the fingerprints. One of the 5 alternative procedures is executed. In the first procedure, the Resource 1 tests the blood. In the second procedure, the Resource 1 tests the hearing. In the third procedure, the Resource 1 checks the eyes and then tests the color vision. In the fourth procedure, the Resource 1 checks the weight. In the fifth procedure, the Resource 1 tests the urines. In any case, the Resource 1 holds the doctor examination. In any case, the subprocess ends. In any case, the process ends.
